# Laced Backend

Laced is a production-ready AI content generation platform with Whisper integration.

## Overview
- Backend built with FastAPI
- PostgreSQL + SQLAlchemy
- Redis + Celery for async tasks
- Modular architecture with services, controllers, schemas
- Full unit and integration test coverage
- Production-ready configuration, logging, and security

## Features
- User authentication & authorization
- Image upload, processing, watermarking, upscaling
- LoRA model training and management
- Email notifications and scheduled jobs
- Real-time WebSocket support
- Fully tested, async-ready backend
