\# Whisper Bot



Whisper is a dynamic Discord bot with personality, emotional responses, and integration with Laced image generator.



\## Setup



1\. Copy `.env.example` to `.env` and fill in your credentials.

2\. Install dependencies: `pip install -r requirements.txt`

3\. Run the bot: `python main.py`



\## Features



\- Slash commands: `/prompt`, `/persona`, `/mood`, `/chat`, `/generate`

\- Multiple personas: base, seductive, playful, romantic, custom

\- Emotional responses

\- Laced integration for image generation

\- Conversation memory per user
